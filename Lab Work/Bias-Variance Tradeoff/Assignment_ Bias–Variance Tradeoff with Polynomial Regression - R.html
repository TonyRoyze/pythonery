Assignment: Bias–Variance Tradeoff with Polynomial Regression

Background

You will analyze a synthetic regression dataset generated from the true function

y=sin⁡(x)+0.1x+ε,ε∼N(0,0.52).

You’ll fit polynomial models of increasing degree, visualize the fits against the true function, and examine the bias–variance tradeoff via train/test MSE.

Data (Run This First)

Use the following R code to generate the dataset and split it into training and test sets:

# --- Data generation and split (R) ---
set.seed(123)
X <- seq(0, 10, length.out = 50)
y_true <- sin(X) + 0.1 * X
y_noisy <- y_true + rnorm(length(X), mean = 0, sd = 0.5)  # sd aligns with Background

# Train/test split (70/30)
set.seed(42)
n <- length(X)
test_idx  <- sample.int(n, size = floor(0.3 * n), replace = FALSE)
train_idx <- setdiff(seq_len(n), test_idx)

train_df <- data.frame(x = X[train_idx], y = y_noisy[train_idx])
test_df  <- data.frame(x = X[test_idx],  y = y_noisy[test_idx])
